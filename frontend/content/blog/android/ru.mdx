# Протоколы АИ на Android

Протоколы АИ — это приложение, которое с помощью ИИ превращает записи встреч в аккуратные протоколы.

Версия для Android уже в Google Play. Попробуйте, если вы ищете приложение для протоколов с ИИ на Android.
