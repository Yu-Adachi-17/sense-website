# Minutes.AI arrive sur Android

Minutes.AI est une application qui transforme vos enregistrements de réunion en comptes rendus clairs grâce à l'IA.

La version Android est maintenant sur Google Play. Essayez-la si vous cherchez une application de compte rendu avec IA sur Android.

[Télécharger sur Google Play](https://play.google.com/store/apps/details?id=world.senseai.minutes)
