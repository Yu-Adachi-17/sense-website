Protokoll KI für iOS hat eine neue Funktion für den automatischen E-Mail-Versand von Besprechungsprotokollen erhalten.

Nachdem Sie die Aufnahme beendet haben, können Sie die App schließen. Sobald das Protokoll fertig ist, wird es automatisch von no-reply@mg.sense-ai.world an die von Ihnen hinterlegten Adressen gesendet.
