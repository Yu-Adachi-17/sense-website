## Rilis
Minutes.AI kini mendukung **rapat daring**. **Klik “Online” → buat URL → bagikan**, dan mulai rapat ala Zoom seketika. Saat rapat berakhir, **notulen dibuat otomatis**, dengan keputusan dan aksi yang jelas.

> Durasi rapat bergantung pada paket dan kuota tersisa Anda.

## Cara memulai
1. Klik **“Online”** di aplikasi atau web  
2. **Buat** URL rapat  
3. **Bagikan** URL ke peserta  
4. Akhiri rapat → **notulen otomatis**

## Keunggulan
- UX **mirip Zoom** (berbasis LiveKit)
- **Satu klik** untuk membuat URL
- **Notulen otomatis** setelah rapat
- **Pilihan format** untuk hasil rapi
- **Multibahasa** untuk tim global

## Catatan
- Batas durasi dan sisa waktu mengikuti **paket/kuota**.  
- Jaringan dan perangkat dapat memengaruhi kualitas.
