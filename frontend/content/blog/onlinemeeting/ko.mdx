## 출시 안내
Minutes.AI가 **온라인 미팅**을 지원합니다. **“Online” → URL 발급 → 공유**만으로 즉시 Zoom 스타일 회의를 시작하세요. 회의가 끝나면 **회의록이 자동 생성**되어 결정 사항과 액션 아이템을 명확히 정리합니다.

> 사용 가능한 시간은 요금제와 잔여 시간에 따라 달라집니다.

## 시작 방법
1. 앱 또는 웹에서 **“Online”** 클릭  
2. **회의 URL 발급**  
3. 참가자에게 **URL 공유**  
4. 회의 종료 → **자동 회의록**

## 특징
- **Zoom 유사** UX (LiveKit 기반)
- **원클릭** URL 발급
- 종료 즉시 **자동 회의록**
- **포맷 선택**으로 가독성 향상
- **다국어 지원**으로 글로벌 협업

## 안내
- 시간 제한과 잔여 시간은 **요금제/쿼터**에 따릅니다.  
- 네트워크와 기기 설정에 따라 품질이 달라질 수 있습니다.
